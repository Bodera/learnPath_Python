```python
def multiplicarse(a, b)
	a * b
```

Toda función que recibe un parámetro obligatoriamente deberá terminar utilizando la claúsula de retornopara que sea posible manipular el resultado de la función. De esa forma:

```python
def multiplicarse(a, b)
	return a * b
```
